# Color Picker

A simple and modern color picker built with HTML, CSS, and JavaScript. It allows users to select a color, preview it in real time, view its HEX and RGB values, and copy them to the clipboard.

## Features

- Real-time color preview
- Native HTML color picker
- HEX color value display
- RGB color value display
- Copy HEX value to clipboard
- Copy RGB value to clipboard
- Clean and responsive interface
- Built with vanilla HTML, CSS, and JavaScript

## Project Structure

```text
color-picker/
├── index.html
├── style.css
└── script.js
```

## Usage

1. Open `index.html` in your web browser.
2. Select a color using the color picker.
3. View the corresponding HEX and RGB values.
4. Copy either value using the provided buttons.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)

## Browser Support

The project works in all modern browsers that support:

- HTML5 Color Input
- Clipboard API
